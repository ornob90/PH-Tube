  • Qns1: Discuss the scope of var, let, and const
  • The scope of var is functional scope, which means it can be accessible from anywhere in the code, even if it's declared inside a function. It works like a global variable.
    On the other hand, let and const are block-scoped, which means the variables declared with let or const are only accessible inside the scope where they are declared.

  • Qns3: What do you mean by REST API?
  • A REST API, or Application Programming Interface, refers to a set of rules used to connect the frontend to the backend or to connect different apps created using various languages. It can be used to access data from a backend. Many websites store their data in JSON format, which can be accessed via an API.
